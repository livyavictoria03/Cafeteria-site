Projeto de site institucional para uma cafeteria fictícia chamada Café Aroma, desenvolvido com HTML, CSS e JavaScript como parte de estudos em desenvolvimento web.

## Descrição do Projeto

O site Café Aroma apresenta informações sobre a cafeteria, incluindo:
- Galeria de imagens com destaque
- Cardápio com produtos e preços
- Seção “Sobre”
- Informações de contato

O objetivo do projeto é aplicar conceitos de estrutura HTML semântica, estilização com CSS e interatividade com JavaScript.
